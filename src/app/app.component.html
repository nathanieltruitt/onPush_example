On Push Example

Open your console

Notice that for personal tasks only three entries are logged into the console, this is because app-personal-input is set to OnPush change detection strategy which means the change detection only runs when any @Input() decorator properies change on the component. Notice how when you type into either input box change detection entries log in the console for every character change for the list on the right. This is because change detection is sent down from the root component to the children.
